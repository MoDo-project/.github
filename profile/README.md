# MoDo - 목표 기반 todo 서비스

## 🌟 About MoDo

MoDo는 '왜'에서 시작하는 목표 지향적 ToDo 서비스입니다. 목표를 먼저 세우고 할 일을 구조화함으로써 더 의미있고 효율적인 실행을 돕습니다. 또한 실시간 공유 기능으로 함께하는 사람들과 서로 자극받으며 성장할 수 있는 환경을 제공합니다.

## 🎯 핵심 가치

<h3>🏁 목표 우선 접근 (Goal-First Approach)</h3>
<p>
  <ul>
    <li>할 일을 추가하기 전에 먼저 목표를 설정하도록 유도</li>
    <li>'무엇을 할까'가 아닌 '왜 해야 하는가'에서 출발</li>
    <li>계층적 목표 구조를 통해 큰 그림과 세부 실행을 연결</li>
  </ul>
</p>

<h3>모든 목표를 달성하는 Motivation (MoDo)</h3>
<p>
  <ul>
    <li>명확한 목표 의식을 바탕으로 한 우선순위 설정</li>
    <li>목적 없는 태스크 남발 방지</li>
    <li>의미 있는 실행으로 이어지는 계획 수립</li>
  </ul>
</p>

<h3>모두 함께 성장하는 환경 (MoDo Together)</h3>
<p>
  <ul>
    <li>실시간 ToDo 공유를 통한 동료들과의 상호 자극</li>
    <li>서로의 진행 상황을 확인하며 동기부여 강화</li>
    <li>혼자가 아닌 '모두 함께' 목표를 달성하는 경험</li>
  </ul>
</p>
